# Narrative Style Cards
Each card follows a consistent 4-section format:

## Card Format
```
[Title]
-----------------
Core Concept:
[One-line description]

Best For:
[3 key use cases]

Key Elements:
[3 essential components]
```

## Cards

-----------------

• Mystery games
